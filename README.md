## Hosted on GitHub Pages

This repo is set up to be published as a GitHub Page. To enable:

1. Create a new repo on GitHub named **`star-citizen-rare-armor`** (same name = correct URL).
2. Push this folder:
   ```bash
   cd "path/to/star-citizen-rare-armor"
   git remote add origin https://github.com/YOUR_USERNAME/star-citizen-rare-armor.git
   git branch -M main
   git push -u origin main
   ```
3. On GitHub: **Settings → Pages** → Source: **Deploy from a branch** → Branch: **main** → Folder: **/ (root)** → Save.
4. After a minute or two, the site will be at `https://YOUR_USERNAME.github.io/star-citizen-rare-armor/`.

## Updating the site

The site is a static build. To update content, edit the source in the `rare-armor-guide` project (parent folder), rebuild with base `'/star-citizen-rare-armor/'`, then copy the new `dist` contents here and push.
